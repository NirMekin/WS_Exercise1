API - VOD Web Service - Genres

Get receive all Genres data:

ws-excercise-vod-genres.herokuapp.com/getAllGenres
no parameters required

POST receive specific Genre data:

Select one below and submit to see example

Comedy





ws-excercise-vod-genres.herokuapp.com/getOneGenre
require parameter: 'cat'

Get receive Movies and Tv Shows by year and rating:

ws-excercise-vod-genres.herokuapp.com/getListYearAndRating/:year/:rating
require parameters: 'year' & 'rating'

notes:

 1. movies with rating equal or higher to parameter 'rating' and their
  year equal to 'year' parameter will be return

 2. Tv Shows with rating equal or higher to parameter 'rating' and their
  year of broadcast are in the range of 'year' parameter will be return

 3. Tv-Shows with 'end_year' = 0 indicates they still broadcast up today
