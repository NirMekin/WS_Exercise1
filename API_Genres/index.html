API - VOD Web Service - Genres

about

GET Return all Genres data:

ws-excercise-vod-genres.herokuapp.com/getAllGenres
no parameters required

POST Return specific Genre data:

require parameter: 'cat'
ws-excercise-vod-genres.herokuapp.com/getOneGenre

Select one below and submit to see example

Comedy





GET Return list of Movies or Tv Shows of specific Genre:

ws-excercise-vod-genres.herokuapp.com/getListofGenre/:genre/:tv_shows (or :movies)
require parameters: 'genre' & movies/tv_shows
